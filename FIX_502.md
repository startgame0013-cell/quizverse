# حل 502 Bad Gateway

## المشكلة
الـ backend على Render يرجع 502 بسبب عدم الاتصال بـ MongoDB.

## الحل (خطوة واحدة إجبارية)

### 1. أضف IP في MongoDB Atlas

**الرابط المباشر:**  
ادخل إلى: **https://cloud.mongodb.com**

ثم:
1. اختر مشروعك (Project 0)
2. من القائمة اليسرى: **Security** → **Network Access**
3. اضغط **ADD IP ADDRESS**
4. اختر **ALLOW ACCESS FROM ANYWHERE** (أو اكتب `0.0.0.0/0`)
5. اضغط **Confirm**

### 2. إعادة النشر على Render

- ادخل **Render** → مشروع quizverse
- اضغط **Manual Deploy** → **Deploy latest commit**

### 3. انتظر 2–3 دقائق

بعدها يجب أن يعمل الموقع.
