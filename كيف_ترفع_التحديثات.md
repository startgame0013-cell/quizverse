# كيف ترفع التحديثات عشان تظهر على الموقع

الموقع العام: **https://quizverse-lyart.vercel.app**  
يأخذ التحديثات من GitHub. أي تعديل على جهازك ما يظهر على الموقع إلا بعد الرفع.

---

## الطريقة (من الطرفية / Terminal)

افتح الطرفية في مجلد المشروع ثم:

```bash
# 1) إضافة كل الملفات المعدلة
git add .

# 2) عمل commit مع رسالة
git commit -m "زر الرجوع + Terms + FAQ + PWA + تحسينات"

# 3) رفع إلى GitHub
git push origin main
```

بعد الـ push، ادخل على [Vercel](https://vercel.com) → مشروعك → انتظر الـ Deploy (1–2 دقيقة). ثم حدّث الصفحة في المتصفح.

---

## من Cursor / VS Code

1. اضغط أيقونة **Source Control** (أو `Ctrl+Shift+G` / `Cmd+Shift+G`).
2. اكتب رسالة في مربع **Message** مثل: `زر الرجوع وتحديثات`.
3. اضغط **✓ Commit**.
4. اضغط **Sync** أو **Push** لرفع التعديلات إلى GitHub.

---

## إذا طلب منك Git اسمك أو البريد

مرة واحدة فقط:

```bash
git config --global user.name "اسمك"
git config --global user.email "your@email.com"
```

ثم أعد الـ commit والـ push.
